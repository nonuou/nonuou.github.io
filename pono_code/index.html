Q

駐車場は何台ありますか？

A

3台ございますが、お席と同時にご予約の方優先のため、当日のご来店では停められない場合もございます。近辺に有料駐車場もありませんのでお車でご来店の方は駐車場もお早めにご予約ください。

Q

犬を連れての食事は可能ですか？

A

テラス席に限り可能で、お席は少ないのためお早めのご予約をお勧めしております。但し、雨天時はキャンセルになりますのでご了承ください。

Q

小さな子供を連れて食事は可能ですか？

A

もちろん可能です。キッズメニューのご用意もあります。

Q

ランチでお酒を出していただくことは可能ですか？

A

ランチメニューには載せておりませんが、お店に飾ってあるお酒はご提供可能です。お気軽にお声かけください。

Q

アレルギー対応はどうなっていますか？

A

可能な限りご対応しますのでご相談ください。但し、使用しているお米に含まれるため小麦アレルギーのみ対応不可になっております。
